The AJAX Tag Library is a set of JSP tags that simplify the use of Asynchronous JavaScript and XML (AJAX) technology in JavaServer Pages. It is based on "Prototype":http://prototype.conio.net/ . Documentation for Prototype can be found "here":http://blogs.ebusiness-apps.com/jordan/pages/Prototype%20Library%20Info.htm and "here":http://www.sergiopereira.com/articles/prototype.js.html .

Find it here: "AJAXTags":http://ajaxtags.sourceforge.net/
